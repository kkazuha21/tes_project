## 🚀 快速部署指南

### 环境要求
- **Java**: JDK 21+
- **MySQL**: 8.0+
- **Web服务器**: Nginx / Apache（推荐）或简单HTTP服务器

### 部署步骤

#### 1. 数据库准备
```sql
CREATE DATABASE sharedlab CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```
然后执行 `database-init.sql` 初始化数据库。

#### 2. 后端部署
1. 修改 `backend/application.properties` 配置数据库连接
2. 运行命令：
   ```bash
   java -jar backend/shared-lab-platform-1.0.0.jar
   ```
3. 验证：访问 http://localhost:8080/api

#### 3. 前端部署

**方式1: Nginx（推荐）**
```nginx
server {
    listen 80;
    server_name your-domain.com;
    
    root /path/to/frontend;
    index index.html;
    
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    location /api {
        proxy_pass http://localhost:8080;
    }
}
```

**方式2: 简单HTTP服务器**
```bash
cd frontend
python -m http.server 3000
```

## 📞 注意事项

1. **数据库配置**: 部署前务必修改 `application.properties` 中的数据库连接信息
2. **JWT密钥**: 生产环境请修改JWT密钥（至少256位）
3. **端口配置**: 默认后端端口8080，可在配置文件中修改
4. **HTTPS**: 生产环境建议配置HTTPS证书
5. **性能优化**: 可根据服务器配置调整JVM参数和数据库连接池
