# 共享实验室平台 - 启动说明

## 📁 项目结构
```
SharedLab/
├── backend/          # 后端 (Spring Boot)
├── frontend/         # 前端 (Vue 3 + Vite)
├── database/         # 数据库脚本
├── 启动说明.md       # 本文件
└── 快速启动.bat      # 一键启动脚本
```

## 🚀 快速启动

### 方法一: 使用一键启动脚本
双击运行 `快速启动.bat`

### 方法二: 手动启动

#### 1. 启动后端
打开 PowerShell:
```powershell
cd C:\SharedLab\backend
.\start-backend-jdk21.bat
```
等待 30-40 秒,看到 "Started SharedLabApplication" 表示启动成功

#### 2. 启动前端
打开另一个 PowerShell:
```powershell
cd C:\SharedLab\frontend
npm run dev
```
等待 10-15 秒,看到 "VITE ready" 表示启动成功

## 🛠️ 常见问题

### 端口被占用
```powershell
# 查看占用端口的进程
netstat -ano | findstr "8080"
netstat -ano | findstr "5173"

# 结束进程 (替换 PID 为实际进程ID)
Stop-Process -Id PID -Force
```

### 重置数据库
在 MySQL 客户端执行:
```sql
USE shared_lab;
source C:/SharedLab/database/init.sql
```
