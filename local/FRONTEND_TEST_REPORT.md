### 路由系统

**当前路由结构:**
```
/login                    - 登录页
/student                  - 学生主界面
  ├─ /devices            - 设备列表
  ├─ /devices/:id        - 设备详情
  ├─ /bookings           - 我的预约
  └─ /tickets            - 我的报修

/teacher                  - 教师主界面
  ├─ /devices            - 设备列表
  ├─ /devices/:id        - 设备详情
  ├─ /bookings           - 预约管理
  ├─ /tickets            - 报修管理
  └─ /my-bookings        - 我的预约

/admin                    - 管理员主界面
  ├─ /dashboard          - 仪表盘
  ├─ /users              - 用户管理
  ├─ /devices            - 设备管理
  ├─ /bookings           - 预约管理
  ├─ /tickets            - 报修管理
  ├─ /categories         - 分类管理
  └─ /logs               - 系统日志

/                         - 根据登录状态自动重定向
```

## 已知问题

### 1. 根路径 404
- **现象**: 访问 `http://localhost:5173/` 返回 404
- **解决方法**: 使用 `http://localhost:5173/index.html` 或直接访问登录页
- **影响**: 轻微，用户通常通过书签或直接输入完整 URL 访问

### 2. Admin 子页面未实现
- **现象**: Admin 菜单的多个选项都指向同一个占位组件
- **受影响**: 用户管理、预约管理、分类管理、系统日志
- **优先级**: 中等，需要后续开发专门的管理页面
