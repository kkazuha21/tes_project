## ❗ 注意事项

1. **时间必须是未来时间**
   - 不能预约过去的时间
   - 不能预约当前正在进行的时间

2. **结束时间必须晚于开始时间**
   - 至少间隔几分钟

3. **避开已有预约**
   - 仔细查看"已有预约时段"列表
   - 选择完全不重叠的时间

4. **预约自动确认**
   - 学生预约后状态直接为"已确认"
   - 无需等待审批

## 🛠️ 故障排查

### 问题：一直显示"该时间段已被预约"

**原因**：选择的时间与已有预约冲突

**解决**：
1. 向下滚动查看"已有预约时段"
2. 选择列表中**没有**的时间
3. 或者换一个设备

### 问题：提交后没有反应

**原因**：可能是网络问题或 Token 过期

**解决**：
1. 按 F12 打开浏览器控制台
2. 查看 Console 是否有错误
3. 查看 Network 标签，bookings 请求的状态码
4. 如果是 401，重新登录

### 问题：预约成功但看不到

**原因**：页面没有刷新

**解决**：
1. 刷新页面
2. 或者点击左侧"我的预约"菜单
3. 应该能看到刚创建的预约
